## Retrospective — 2026-03-20 (Recovered)

### Lessons Learned
1. **Governance Boosting (SQL Rank Multiplication)**: Verified that applying a multiplier to core dogma files (`AGENTS.md`, `MANIFESTO.md`) during SQLite retrieval ensures they consistently occupy the `top-k 1` slot, preventing critical guardrails from being displaced by high-volume research noise.

4. **Benchmark Script Evolution**: `scripts/benchmark_rag.py` must support `--template-path` to test prompt-scaffolding as a primary variable, as prompt structure is a stronger driver of accuracy than model size in the 3B–8B range.
